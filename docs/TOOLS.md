# Nexus-USB Tools Reference

## Security & Malware Scanning

### ClamAV
Open-source antivirus engine for detecting trojans, viruses, malware, and other threats.
- Full system scan
- Quick scan of common locations
- Update virus definitions

### chkrootkit
Checks for signs of rootkits on the local system.

### rkhunter
Scans for rootkits, backdoors, and possible local exploits.

## System Recovery

### TestDisk
Powerful tool to recover lost partitions and make non-booting disks bootable again.

### PhotoRec
File data recovery software designed to recover lost files from hard disks, CD-ROMs, and lost pictures from digital camera memory.

### ddrescue
Data recovery tool that copies data from one file or block device to another, trying to rescue data in case of read errors.

### fsck
File system consistency check and repair tool for Linux filesystems.

## Disk Management

### GParted
Graphical partition editor for creating, reorganizing, and deleting disk partitions.

### fdisk
Command-line partition table manipulator.

### smartmontools
Controls and monitors storage systems using SMART (Self-Monitoring, Analysis and Reporting Technology).

## Password Tools

### chntpw
Utility to reset Windows user passwords without knowing the old password.

### ophcrack
Windows password cracker based on rainbow tables.

## Network Tools

### Wireshark
Network protocol analyzer for troubleshooting and analysis.

### nmap
Network exploration tool and security scanner.

### netcat
Networking utility for reading/writing network connections.

### tcpdump
Command-line packet analyzer.

## System Information

### hwinfo
Hardware information tool that provides detailed info about hardware.

### lshw
Lists detailed information on hardware configuration.

### memtest86+
Memory testing tool to check RAM for errors.
